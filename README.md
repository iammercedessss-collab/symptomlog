# SymptomLog 🩺
A personal health symptom tracker built in Python.

## What it does
- Log symptoms with severity (1–10) and notes
- View your full history in a clean table
- See patterns — most frequent symptoms, average severity over time
- Search any symptom to see its history

## Why I built it
As a Human Anatomy student, I wanted a tool that treats health data the way the body actually works — as something that changes over time, not a single snapshot. Health signals are temporal, and this tool captures that.

## How to run it
Make sure Python is installed, then:
\`\`\`
python symptomlog.py
\`\`\`
No external libraries needed
